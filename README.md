# california-housing-ml
Machine learning project for housing price prediction and classification
# Predicción de Precios de Vivienda en California

Este proyecto utiliza modelos de *machine learning* para analizar y predecir el valor medio de viviendas en distritos censales de California a partir de variables demográficas, geográficas y de vivienda.

## Objetivo
- Predecir el valor medio de las viviendas (regresión)
- Clasificar zonas en categorías de precio: Bajo, Medio, Alto y Lujo

El enfoque está en generar insights útiles para la toma de decisiones, no solo en la precisión del modelo.

## Dataset
California Housing Dataset (Censo de EE. UU., 1990).  
Aunque es histórico, refleja factores estructurales que siguen influyendo en el mercado inmobiliario actual.

## Modelos
- Regresión Lineal  
- Árbol de Decisión  
- Random Forest  
- XGBoost (mejor desempeño)

## Herramientas
Python, Pandas, NumPy, Scikit-learn, XGBoost
